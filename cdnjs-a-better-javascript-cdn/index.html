1. CDNJS: A Better JavaScript CDN

  2. Quick review

    JavaScript CDNs provide hosted copies of popular scripts.

  3. The value proposition

    1. Some guy builds site with jQuery from JavaScript CDN Inc.
    2. User visits that site and downloads jQuery
    3. You build site with jQuery from JavaScript CDN Inc.
    4. User visits your site and doesn't have to re-download jQuery
    5. Profit (literally)!

  4. A hot cache is a happy cache

    JavaScript CDNs allow us to use scripts that will usually be hot in cache.

  5. Existing JavaScript CDNs

    There are a few..

    • Google Libraries
    • Microsoft AJAX CDN
    • Yahoo! YUI Hosting
    • etc.

  6. ...they host some libraries...

    • jQuery
    • jQuery UI
    • Prototype
    • ExtJS
    • MooTools
    • Scriptaculous
    • YUI
    • Some others..

  7. ...and have some problems

    • Existing CDNs only host "major" libraries, and update their selection infrequently.
    • Up-and-coming libraries (backbone, underscore, raphael) are never offered.
    • There is no official or effective channel for users to propose amendments to the selection.

  8. Enter CDNJS

    A JavaScript CDN for the people, by the people!

  9. CDNJS hosts major libraries

    You'll find your jQueries, YUIs and MooTools when you need them

  10. CDNJS also hosts the other stuff

    • labjs
    • requirejs
    • yepnope
    • raphael
    • underscore
    • backbone
    • and many more...

  11. CDNJS is community maintained

    • To contribute, fork on Github
    • Pull requests accepted based on community support
    • Community support channel via uservoice

  12. CDNJS is powered by CloudFlare

    We are dedicated to providing CDNJS with hosting across our own content delivery network, with 12 locations globally and growing.

  13. CDNJS is fast, everywhere, always

    Informal tests suggest up to twice as fast as Google's JavaScript CDN!

    CDNJS probably contains a library you're using, so consider it for your next project!
